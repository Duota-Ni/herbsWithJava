# herbsWithJava
## Java Web
使用MVC模式开发草药管理系统  
软件：Eclipse IDE+Tomcat+MySQL  
技术：Java+HTML+CSS+JavaScript+jdbc+关系型数据库  

### 系统设计框架 
HTML+JSP+CSS实现系统的视图层  
servlet+javascript实现逻辑设计和控制  
在实现数据库连接时采用JDBC访问数据库  
同时实现数据分页，出库订单，使用了请求作用域、会话作用域、应用作用域。
